Education

Intensive bootcamps, workshops, and training in AI, machine learning, bioinformatics, and scientific computing.

Education Program

ESB AI Lab offers intensive, hands-on educational programs in AI, machine learning, bioinformatics, scientific computing, and related fields. All programs are taught in person in San Diego by Dr. Edwin Solares (computational biologist, former UC San Diego and UC Davis faculty) with access to research-grade hardware and compute infrastructure.

Programs are designed for working scientists and researchers. No prior programming experience is required for introductory offerings.

Bootcamp Programs

Bootcamps deliver 32 contact hours of hands-on instruction. Schedule formats are flexible: weekday intensives, weekend intensives, half-day sessions, or evening series. All bootcamps include an ESB AI Lab laptop pre-configured with required software, compute infrastructure access, meals during sessions, all course materials, and a Certificate of Completion.

Applied Bioinformatics for Genomics Research

Go from raw sequencing data to biological insight using the same tools and pipelines used in active genomics research. Covers Linux command line, sequence alignment, variant calling (GATK), RNA-seq analysis (DESeq2), and reproducible pipelines.

For: Wet-lab biologists, graduate students, postdocs working with sequencing data Prerequisites: None

Python for Data Science and Machine Learning

Build a working foundation in Python, data analysis, and machine learning grounded in real scientific datasets. Covers Python fundamentals through scikit-learn and intro deep learning with PyTorch.

For: Scientists and researchers in any field Prerequisites: None

AI/ML for Biological Applications

Apply modern machine learning and AI methods to real biological problems, from image classification to protein function prediction. Covers computer vision for biology, transfer learning, model deployment, and AI ethics.

For: Biologists and bioinformaticians with basic Python Prerequisites: Basic Python proficiency

Data Science for Conservation and Agriculture

Use data science and AI tools to work with environmental, agricultural, and ecological data. Covers geospatial data, remote sensing, time-series analysis, and edge AI for field sensors.

For: Ecologists, conservation scientists, agricultural researchers Prerequisites: Basic Python or R

Deep Learning for Scientists

Build deep learning models from the ground up, starting with the math, moving through modern architectures, and ending with real applications. Covers NumPy to PyTorch, CNNs, Transformers, and experiment tracking.

For: Scientists who want to deeply understand deep learning Prerequisites: Python proficiency, basic linear algebra

GPU Computing for Scientific Applications

Learn to accelerate scientific computing with GPUs. Covers CUDA programming, Numba, CuPy, RAPIDS (cuDF, cuML), profiling, and multi-GPU setups. Hands-on with ESB AI Lab GPU servers.

For: Computational scientists running numerically intensive workloads Prerequisites: Solid Python proficiency

Embedded AI with NVIDIA Jetson

Build and deploy AI applications on edge devices using NVIDIA Jetson and AGX Orin hardware. Covers JetPack SDK, TensorRT, DeepStream, real-time computer vision pipelines, and power-constrained optimization.

For: Researchers building AI-powered field systems Prerequisites: Basic Python and Linux Cohort cap: 11 participants

Big Data with Apache Spark

Process and analyze datasets that don’t fit on a single machine. Covers PySpark, Spark SQL, MLlib, Spark Structured Streaming, performance tuning, and AWS EMR deployment.

For: Researchers whose data has outgrown single-machine tools Prerequisites: Solid Python proficiency

Distributed Computing with Ray

Scale Python and ML workflows from a laptop to a cluster without rewriting code. Covers Ray Core, Ray Data, Ray Train, Ray Tune, Ray Serve, and Kubernetes deployment.

For: Researchers running computationally intensive Python workflows Prerequisites: Strong Python, basic ML

Cloud Computing with AWS for Researchers

Learn Amazon Web Services for research workloads. Covers EC2, S3, Lambda, Batch, SageMaker, cost management, and reproducible research environments. Cloud credits provided.

For: Researchers who need more compute than their laptop provides Prerequisites: Basic Linux

Cloud Computing with GCP for Researchers

Learn Google Cloud Platform for research. Covers Compute Engine, BigQuery, Terra/Cromwell, Vertex AI, and cost optimization. Cloud credits provided.

For: Researchers, especially in genomics and life sciences Prerequisites: Basic Linux

Containers for Scientists

Master containers from first Dockerfile to production-grade deployment. Covers Docker, Compose, Singularity/Apptainer for HPC, networking, storage, registries, CI/CD, and reproducible research pipelines.

For: Researchers using or wanting to use Docker Prerequisites: Basic Linux

Networking for Scientists

Understand how computer networks actually work. Covers TCP/IP, DNS, SSH, firewalls, VPNs, network troubleshooting, lab server configuration, and cloud networking basics.

For: Researchers managing lab servers or remote access Prerequisites: Basic Linux

Advanced Containers: Building Research Infrastructure

Build production-grade containerized services. Covers reverse proxies (Nginx/Traefik), authentication (OAuth2/SSO), API gateways, TLS/SSL, logging and monitoring, and security hardening.

For: Researchers building multi-service infrastructure Prerequisites: Docker experience

Workshop Programs

Workshops are shorter, focused programs on a specific skill or topic. In-person in San Diego, ESB AI Lab laptops provided, materials and Certificate of Completion included.

Linux for Scientists (8 hours)

Learn the command line from scratch: navigation, text processing with grep/sed/awk, shell scripting, SSH, and Git basics. The recommended starting point before most bootcamp programs.

For: Scientists who have never used a terminal Prerequisites: None

Introduction to Containers and Docker (8 hours)

Get started with Docker: pull images, run containers, write Dockerfiles, Docker Compose basics, and Singularity/Apptainer for HPC.

For: Researchers who want reproducible environments Prerequisites: Basic Linux

Docker for Reproducible Research (8 hours)

Design container strategies for making computational research reproducible: version-pinning, packaging pipelines, Zenodo archiving, and long-term reproducibility planning.

For: Researchers who know Docker basics Prerequisites: Basic Docker

Introduction to Genetics and Genomics (8 hours)

A biology primer for computational people. Covers molecular biology fundamentals, genome organization, sequencing technologies, key genomics concepts, and current frontiers.

For: Data scientists and computational researchers entering genomics Prerequisites: None

HPC and Cloud Computing for Research (8 hours)

Understand the research computing landscape. Covers Slurm, batch jobs, parallelization, and intro to AWS/cloud cost optimization.

For: Researchers who need more computing power Prerequisites: Basic Linux

Building AI Chatbots for Research (4 hours)

Build a domain-specific AI assistant using LLM APIs, prompt engineering, and retrieval-augmented generation (RAG) for your own documents.

For: Researchers wanting to build useful AI tools for their group Prerequisites: Basic Python

3D Printing for Science and Prototyping (8 hours)

Design, slice, and print functional objects for your lab. Covers CAD (OpenSCAD, FreeCAD), FDM printing, material selection, and design for scientific applications.

For: Researchers who want custom lab equipment or field housings Prerequisites: None

Robotics for Scientific Data Collection (8 hours)

Build sensor systems and automated data collection platforms using Arduino, Raspberry Pi, and scientific sensors. Build a working prototype you take home.

For: Field researchers and lab scientists wanting custom data collection Prerequisites: None

Accessibility and Scholarships

ESB AI Lab is committed to broadening scientific participation. All programs include a published sliding-scale fee structure and scholarship provisions for participants from underserved communities and minority-serving institutions.

Many employers cover professional development costs. We accept institutional purchase orders (net 30) and provide all documentation your HR or finance department needs for reimbursement.
